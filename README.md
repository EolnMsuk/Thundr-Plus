# Thundr-Plus
A Tampermonkey script which bypasses face detection on thundr.com

## How to Install & Use the Thundr-Plus Tampermonkey Script

### 1️⃣ Install Tampermonkey

**Install it for your specific browser:**

* **Chrome / Chromium / Brave / Edge:**
  [https://tampermonkey.net](https://tampermonkey.net)
* **Firefox:**
  [https://addons.mozilla.org/firefox/addon/tampermonkey/](https://addons.mozilla.org/firefox/addon/tampermonkey/)

After installing, you should see a **Tampermonkey icon** in your browser toolbar.

### 2️⃣ Add the Thundr-Plus script

1. Click the **Tampermonkey icon**
2. Select **“Create a new script…”**
3. A script editor will open with some default code

### 3️⃣ Paste the Script (From Plain Text)

1. **Delete everything** in the editor
2. **Paste the full script from**: https://raw.githubusercontent.com/EolnMsuk/Thundr-Plus/refs/heads/main/Thundr-Plus-Script
4. Press **Ctrl + S** (or **File → Save**)

You should see a confirmation that the script is saved and enabled.

### 4️⃣ Enable the Script

1. Click the **Tampermonkey icon**
2. Make sure the script is **enabled** (toggle should be ON / green)
3. Also make sure **Tampermonkey itself is enabled**

### 5️⃣ Use the Script

1. Go to https://thundr.com/
2. **Refresh the page**
3. The script should now run automatically

---

### Troubleshooting (If It Doesn’t Work)

* Refresh the page after installing
* Make sure you’re on the **correct website**
* Check that the script is **enabled**
* Press **F12 → Console** and look for errors
* Some sites block scripts unless permissions are allowed

### Removing the Script (To Remove)

* **Edit:** Tampermonkey icon → Dashboard → Click the script
* **Disable:** Toggle it off
* **Delete:** Dashboard → Trash icon
